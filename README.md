1. Install Dist::Zilla: `cpanm Dist::Zilla`
1. Install dependencies: `dzil listdeps --missing --author --develop | cpanm`
1. Run smoke tests: `dzil smoke --release --automated --author`
1. Install module locally: `dzil install`
